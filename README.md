Real-time air quality monitoring using IoT sensors and machine learning. The system automatically activates an air purifier when pollution levels are high.

Tech Stack:
Programming: Python
ML: Decision Tree
Hardware: ESP32, MQ2/MQ3/MQ7, Temp & Humidity Sensor
Cloud: Firebase
Visualization: Power BI

Features:
Real-time pollution monitoring
Cloud storage & dashboard
Pollution prediction using ML
Automatic air purifier activation
